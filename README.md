# Home Router

## Features

- Setup PiHole or something similar
- LAN domain name resolution
- CA for internal SSL certificates
- Jellyfin
- Sonarr / Radarr

## Usage

Run this ansible script, then set the DNS server on your ISP router to the computer running this.
